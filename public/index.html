AC Controller

Project Describtion:

A web application to control an air-cooler (AC) through an infrared emitter that is placed in front of the AC and connected to an intel galelio board, the board itself is connected to the web application through a web interface.


Objectives:

  • Control the AC unit through a web application
  • Try to minimize cost, to make it affordable if we want to sell the unit in the future
  • In the future, We plan to increase the number of home devices that can be controlled by the web application.


Main Components:

  • Intel® Quark SoC X1000
  • Wifi module
  • Infrared emitter
